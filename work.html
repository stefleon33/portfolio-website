PromptAlchemy – AI Prompt-Building App (Chingu Voyage Project)

PromptAlchemy is a web application designed to help users write more effective AI prompts using the Pentagram methodology. Built as part of a 4-week remote team collaboration through Chingu, this project gave me hands-on experience with real-world Agile workflows, version control, and collaborative decision-making.

Tech Stack: React · Vite · Google Gemini API · Netlify · Figma

My Contributions:
-Led frontend development using React and Vite, building out the user input forms and response rendering logic
-Integrated the Google Gemini API for dynamic prompt generation and display
-Collaborated on responsive UI design based on our shared Figma prototype
-Managed GitHub workflows and pull requests to ensure clean, maintainable code

Key Features:
-Structured form based on the Pentagram framework for writing prompts
-Real-time AI responses displayed with Markdown formatting
-Smooth animations and mobile-friendly layout
-Deployed live via Netlify with continuous updates during development

Full Stack myFlix App

This single page application is a complete web app (client-side and server-side) that uses a MERN (MongoDB, Express, React, and Node.js) stack. The app uses Parcel as its build tool and is written using the React library and uses Bootstrap as a UI library for styling and responsiveness. React Redux is used for state management.

Users of this app are able to:
-Register, view and update user information (username, password, email, birthday, favorite movies), logout and deregister
-View all movies
-Search for a movie
-Select a single movie for more information
-Add/remove a movie from their list of favorites

Chatter App

This web application was built using React Native. The app provides users with a chat interface and options to share images and their location. The data is stored online as well as offline. The app was developed using Expo, the chat conversations are stored in Google Firestore Database and the images in Firebase Cloud Storage. Chat conversations are also stored locally. The chat interface and functionality were created using the Gifted Chat library.

Users of this app are able to:
-Join the chat room
-Choose the background color for the chat screen before joining the chat
-Send and receive messages, images and their location
-Read their messages offline
-The chat app is compatible with a screen reader

Meet App

This is a progressive web application (PWA) built with React using a test-driven development (TDD) technique. The application uses the Google Calendar API to fetch upcoming events.

Users of this app are able to:
-Filter events by city
-Show/hide event details
-Specify the number of events shown
-Use the app when offline
-Add an app shortcut to their homescreen
-View charts visualizing event details
